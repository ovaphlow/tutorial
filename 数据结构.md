# 数据结构

## 表

### bulletins / 公告、系统通知 等

```sql
CREATE TABLE `bulletins` (
  `id` bigint NOT NULL,
  `time` datetime NOT NULL,
  `title` varchar(50) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL,
  `tags` json NOT NULL,
  `detail` json NOT NULL,
  `expire_time` datetime NOT NULL,
  PRIMARY KEY (`id`),
  KEY `bulletins_time_IDX` (`time`) USING BTREE,
  KEY `bulletins_expire_time_IDX` (`expire_time`) USING BTREE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;
```

### events / 日志、操作、消息 等

```sql
CREATE TABLE `events` (
  `id` bigint NOT NULL,
  `relation_id` bigint NOT NULL,
  `reference_id` bigint NOT NULL,
  `tags` json NOT NULL,
  `detail` json NOT NULL,
  `time` datetime NOT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;
```

### settings / 设置、常量 等

```sql
CREATE TABLE `settings` (
  `id` bigint NOT NULL,
  `root_id` bigint NOT NULL,
  `parent_id` bigint NOT NULL,
  `tags` json NOT NULL,
  `detail` json NOT NULL,
  PRIMARY KEY (`id`),
  KEY `settings_root_id_IDX` (`root_id`) USING BTREE,
  KEY `settings_parent_id_IDX` (`parent_id`) USING BTREE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;
```

### subscribers / 用户

```sql
CREATE TABLE `subscribers` (
  `id` bigint NOT NULL,
  `name` varchar(50) COLLATE utf8mb4_general_ci NOT NULL,
  `tags` json NOT NULL,
  `detail` json NOT NULL,
  `time` datetime NOT NULL,
  PRIMARY KEY (`id`),
  KEY `subscribers_name_IDX` (`name`) USING BTREE,
  KEY `subscribers_time_IDX` (`time`) USING BTREE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;
```
